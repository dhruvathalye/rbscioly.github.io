# RBHS Science Olympiad Website

Static website ready for GitHub + Vercel deployment.

## Files
- `index.html` — website content
- `styles.css` — colors and layout
- `script.js` — mobile menu
- `public/logo.png` — add your logo here

## Deploy to Vercel
1. Push this folder to GitHub.
2. Go to Vercel and import the GitHub repository.
3. Framework preset: **Other** or **Static**.
4. Build command: leave blank.
5. Output directory: leave blank or use `.`.
6. Deploy.

## Edit social links
In `index.html`, replace the `href="#"` values in the Social Media section with your Instagram and Discord links.
